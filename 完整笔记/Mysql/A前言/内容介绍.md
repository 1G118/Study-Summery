# 为什么要学习mysql？
在学习一个知识点之前往往需要搞懂为什么要学习它。用户在使用程序或网站时会生成各种各样的数据，就像打游戏暂停后退出后会自动记录存档一样。这个存档就是我们使用游戏软件时产生的数据，它会被保存在你本地电脑里面的某个文件夹里面。当存档很少时，你可以很轻易的找到你想要的数据，但一旦文档数量增多，数量达到成百上千过万时，在想找到你想要的数据就会耗费大量的时间。而MySQL就可以帮你创建和管理数据，在你想要时可以直接通过搜索快速找到想要的数据。
# 什么是mysql
Mysql是一款介于应用和数据之间，用于存放和管理数据的软件，通过一些设计将数据变成像excel的数据表，提供更加方便的CRUD。
